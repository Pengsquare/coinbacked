# How to build

1. Install packages

```
npm install
```

2. Build 
```
npm run build
```

3. Build production version (minified)

```
npm run build --production
```

4. Build and run local fiddles

```
npm run fiddle
npm run fiddle-web
npm run fiddle-web-start-server
```

# How to use

Use `dist/coinbacked.js` or `coinbacked.min.js` in your project.
