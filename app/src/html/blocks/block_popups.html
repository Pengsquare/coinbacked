About Backing

If you start backing, a new account will be created for this token and backed with an amount of native SOL. This amount can only be retrieved by burning the token.

However owns the token, also owns the backing amount. For fungible tokens, the amount retrieved from backing is proportional to the amount owned.

What is Validation?

You can validate the current backing setup on-chain. The coinbacked contract will be executed and check general settings (e.g., all accounts valid and correctly maintained). This is not a holder account specific validation .

All results will be logged and stored on as transaction log on-chain. Execution will generate a processing fee.
